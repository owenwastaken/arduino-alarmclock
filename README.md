## Required materials
* Arduino with at least 10 digital pins and SCL and SDA pins
* Breadboard (Makes everything a lot easier) (Full size Breadboard recommended)
* Arduino compatible LCD (16x2 Minimum)
* Potentiometer (For controlling LCD contrast)
* 3 Buttons (Could be less if you want to change some code around and sacrifice some features.)
* 3 10K OHM Resistors
* Pico buzzer (For Alarm and UI Clicks)
* DS3231 RTC Module (For an accurate clock)
